Nunchucks

A modern rewrite of Nunjucks focused on composable templates and multi-language support.

Project status: active rewrite, TypeScript-first with Go runtime in progress.

TypeScript Engine

Current engine in src/ with rendering, includes/extends, filters, and precompile flow.

Go Engine

New Go package in go/ with core rendering support for files and strings.

WASM Bridge

Go core compiles to WebAssembly with a JS loader for cross-runtime embedding.

Docs + Pages

Static docs in docs/ deployed through GitHub Pages workflow.

Feature Snapshot

  • {{ expression }} variable rendering
  • {% set %}, {% if/elif/else %}, {% for %}
  • {% include %} and {% extends %} with blocks
  • String filters: lower, upper
